# Flow — Цветочный магазин

Flow — стильный лендинг для цветочного магазина с современным дизайном и удобной навигацией. Сайт создан для демонстрации услуг флористики, портфолио букетов и оформления мероприятий.

## Описание проекта
Стильный лендинг для цветочного магазина Flow с адаптивным дизайном и удобной навигацией.

## ✨ Основные возможности
- Адаптивный дизайн для всех устройств
- Плавная навигация по секциям
- Галерея портфолио букетов
- Форма обратной связи
- Интеграция соцсетей (Telegram, Instagram)

## 📱 Секции сайта
- **Главная** — баннер с призывом к действию
- **О нас** — история магазина
- **Услуги** — букеты и декор
- **Галерея** — 8+ цветочных композиций
- **Контакты** — форма заявок

## 🛠 Стек технологий
- HTML5
- CSS3
- Font Awesome 6.5
